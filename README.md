### Planner-System

### Iniciando
- Java 8
- JSF
- Primefaces
- MySQL
- Netbeans 8.2

### Desenvolvimento
- Importar projeto Java Web
- Criar banco de dados **planner_db**
- Executar script SQL localizado no arquivo Database.sql em **planner/conf**
- Executar projeto e cadastrar um login para acesso
